# 逆战未来枪械应用

一个用于展示和管理逆战游戏枪械信息的Web应用，支持武器信息展示、搜索、复制以及管理员后台管理功能。

## 功能特点

### 主要功能
- 🎯 **武器展示**：以卡片形式展示武器信息，包括名称、皮肤、ID、功能等
- 🔍 **搜索功能**：支持按武器名称搜索
- 📋 **一键复制**：点击复制按钮快速复制武器信息
- 📱 **响应式设计**：适配移动端和桌面端
- 🎨 **现代UI**：采用玻璃态效果和渐变背景，视觉效果美观

### 管理员功能
- 🔐 **管理员登录**：安全的管理员登录系统
- ➕ **添加武器**：支持添加新的武器信息
- ✏️ **编辑武器**：修改现有武器信息
- 🗑️ **删除武器**：删除不需要的武器
- 📤 **数据导出**：将武器数据导出为JSON文件
- 📥 **数据导入**：从JSON文件导入武器数据
- 🔄 **密码修改**：支持修改管理员密码

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 交互功能
- **Tailwind CSS** - 样式框架
- **Font Awesome** - 图标库
- **LocalStorage** - 本地数据存储

## 如何使用

### 直接使用
1. 下载项目文件
2. 打开 `index.html` 文件即可使用

### 部署到服务器
1. 将所有文件上传到Web服务器
2. 通过浏览器访问对应URL

## 数据说明

武器数据存储在浏览器的LocalStorage中，包括以下字段：
- `name` - 武器名称
- `id` - 武器ID
- `skin` - 皮肤名称（可选）
- `function` - 功能或用途
- `type` - 武器类型
- `copyText` - 复制文本
- `fullText` - 完整显示文本
- `description` - 武器描述
